# Amelia's Cats & Kitties Website

Welcome to my personal static website celebrating the "Adopt, Don't Shop" movement and the incredible journey of fostering kittens! 🐾

## Overview
This is a beautiful, responsive, Single Page Application (SPA) built using React, Vite, Tailwind CSS, and daisyUI (using the awesome `cupcake` theme). 

## How It's Made
- **React + Vite**: For a snappy and modern development experience.
- **Tailwind CSS + daisyUI**: For gorgeous styling, beautiful typography, and consistent custom components without the hassle of raw CSS.
- **React Router**: For perfectly seamless client-side navigation between Home, Gallery, and About pages.
- **Fully Static**: Ready to be deployed beautifully without any backend services.

## Getting Started Locally

1. Ensure you have `Node.js` and `npm` installed.
2. Run standard installation inside this directory:
   ```bash
   npm install
   ```
3. Start the Vite development server:
   ```bash
   npm run dev
   ```

## Deployment
This static application is designed to be easily deployed to **Cloudflare Pages**. 
Simply point your Cloudflare dashboard to this repository, set the build output to `dist`, and set the build command to `npm run build`.
